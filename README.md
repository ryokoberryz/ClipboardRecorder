# 剪贴板记录器

一个功能强大的Android剪贴板记录应用，可自动记录所有复制内容，支持批量操作和导出。

## 功能特性

- 📋 **自动记录**：后台自动记录所有剪贴板内容
- ⚙️ **灵活设置**：可设定最大记录条数和保存天数
- 📱 **批量操作**：支持批量选择、复制、删除和导出
- 📤 **批量复制**：多条记录自动换行合并
- 📄 **批量导出**：导出为文本文件，包含时间戳
- 🔍 **搜索功能**：快速查找历史记录
- 🔄 **开机自启**：设备重启后自动启动服务

## 技术实现

- **Kotlin** 开发语言
- **Room** 数据库存储
- **Hilt** 依赖注入
- **前台服务** 确保后台运行
- **Material Design** 现代UI
- **ViewBinding** 高效UI绑定
- **DataStore** 设置存储

## 如何构建

### 前提条件

- Android Studio 2023.1+ 
- JDK 17+
- Android SDK (API 34+)

### 构建步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd ClipboardRecorder
   ```

2. **使用Android Studio打开项目**
   - 启动Android Studio
   - 选择 "Open an existing project"
   - 导航到项目目录并选择

3. **同步Gradle**
   - Android Studio会自动提示同步Gradle
   - 点击 "Sync Now" 按钮

4. **构建APK**
   - 点击菜单栏的 "Build" → "Build Bundle(s) / APK(s)" → "Build APK(s)"
   - 构建完成后，点击 "locate" 按钮找到生成的APK文件

5. **安装到设备**
   - 将生成的APK文件传输到Android设备
   - 在设备上打开文件并安装
   - 首次启动时授予必要的权限

## 如何使用

1. **启动应用**：首次启动时，应用会自动开始监听剪贴板
2. **查看记录**：主界面会显示所有历史记录，按时间倒序排列
3. **批量操作**：
   - 长按任意记录进入选择模式
   - 点击其他记录进行多选
   - 使用顶部工具栏进行批量操作
4. **调整设置**：点击右下角的设置图标，可调整最大记录条数和保存天数
5. **导出记录**：选择需要导出的记录，点击导出按钮

## 权限说明

- **POST_NOTIFICATIONS**：显示前台服务通知
- **RECEIVE_BOOT_COMPLETED**：开机自启
- **FOREGROUND_SERVICE**：保持后台服务运行

## 注意事项

- 应用使用前台服务确保后台运行，这会在通知栏显示一个持续的通知
- 为了确保服务正常运行，请不要在系统设置中禁止应用的自启权限
- 定期清理旧记录可提高应用性能
